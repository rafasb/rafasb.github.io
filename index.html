    Wordpress en docker con certificado Let's Encrypt

    Útil para poner rápidamente en marcha un proyecto wordpress y poder mostrarlo públicamente con seguridad SSL. Hace uso de un proxy inverso para facilitar el acceso mediante cualquier nombre DNS y disponer del certificado gratuido y actualizado del servicio de Let's Encrypt. Al emplear docker facilita el backup y el cambio de proyecto simplemente copiando y restaurando los directorios de datos.
